09/02 — Negotiation Skills

This class was about understanding negotiation and how it can be used to solve problems and reach agreements in both personal and professional life. The lecturer explained different negotiation styles, the importance of listening, and how to find a solution that works for both sides. We also learned that negotiation is not about winning or losing but about creating fair outcomes.

Key points

  • Core negotiation skills
  • Real-life relevance
  • Practical awareness
